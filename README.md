# Auto-Hellas Site

Πρώτη έτοιμη βάση για το site της Auto-Hellas ΓΕΩΡΓΑΤΣΗΣ.

## Το project αυτό έχει:
- premium μαύρο homepage
- φίλτρα αναζήτησης
- κάρτες αυτοκινήτων
- demo panel για προσθήκη αγγελιών
- τοπική αποθήκευση στον browser
- section τοποθεσίας και στοιχεία επικοινωνίας

## Τοπική εκτέλεση
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Σημείωση
Αυτό είναι το πρώτο deployable βήμα. Στο επόμενο στάδιο συνδέεται με Supabase για πραγματικές αγγελίες, login και φωτογραφίες.
